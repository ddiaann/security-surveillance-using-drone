# security-surveillance-using-drone
Source code for multi-camera drone surveillance and zero-shot person re-identification.

Multi-Camera Drone Surveillance System
This repository contains the source code developed for my dissertation on zero-shot person re-identification using OpenCLIP and a DJI Tello drone.

Contents:
- drone_tracking_system.ipynb: The primary notebook containing the logic for initial YOLOv8 detection, OpenCLIP similarity scoring, and the PI flight controller for the drone.
Note: This code was originally executed in Google Colab. Due to the hardware dependencies of the DJI Tello drone and the live IP camera feeds, some blocks require a local runtime environment to execute fully.
